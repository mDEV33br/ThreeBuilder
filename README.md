# TreeBuilder v1.1

Gerador de Estrutura de Pastas e Arquivos - Ferramenta para criar rapidamente estruturas de projetos a partir de árvores de diretórios.

## Funcionalidades

- Editor de texto com barra de rolagem
- Criar estruturas complexas de projetos a partir de árvore de diretórios
- Carregar arquivos `.txt` com a estrutura
- Exemplo de estrutura pronta
- Limpar editor
- Interface GUI interativa e intuitiva
